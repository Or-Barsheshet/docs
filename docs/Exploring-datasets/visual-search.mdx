---
title: "Visual Search"
description: "Find visually similar images, objects, or patterns in your dataset using Visual Layer's powerful similarity search options."
---

Visual Layer enables flexible and intuitive **visual similarity search** across your datasets. You can find near-duplicate images, match specific objects, or search by uploading an image. 

Choose to run similarity searches:

- [on a cluster](#similarity-search-on-a-cluster)  
- [on an image/object](#similarity-search-on-an-image-or-object)  
- [using a region of interest](#similarity-search-using-a-region-of-interest)  
- [via image upload](#similarity-search-via-image-upload)  

## Similarity Search on a Cluster

To find similar results starting from a cluster:

1. Hover over the desired cluster in your dataset.
2. Click **Find Similar** at the bottom of the card.

## Similarity Search on an Image or Object

You can trigger a similarity search: 
- [from the Dataset page](#search-from-the-dataset-page)  
- [from the Cluster Details page](#search-from-the-cluster-details-page)  
- [by object](#search-by-object)

### Search from the Dataset Page

1. Click into a cluster.
2. Hover over the image or object and click **Find Similar**.

<Image src="https://files.readme.io/67833bc9d2959b2165dc17f6c69fa0e6c6eadd09b98f090dba08e0e6ca813a76-Scene-1_2.gif" alt="Similarity Search from Dataset Page" />

### Search from the Cluster Details Page

1. Click an image or object to open the **Image Details Page**.
2. Click **Find Similar**, then click the similarity icon at the center of the image.

<Image src="https://files.readme.io/1e3113710ffe12699bc92844d8bbf90f8a35bcae9c7b9e8da826047ebf5e9a3b-Scene-1_2_copy.gif" alt="Similarity Search from Image Details Page" />

## Similarity Search using a Region of Interest

Search based on a cropped portion of an image:

1. Open an image from any cluster.
2. Click **Find Similar**.
3. Drag to crop the Region of Interest.
4. Click the similarity icon again to start the search.


## Similarity Search via Image Upload

You can also initiate a search using an external image:

1. On the Dataset Page, click the **Find Similar** icon near the search bar.
2. Upload an image from your device.
3. Optionally crop before starting the search.
