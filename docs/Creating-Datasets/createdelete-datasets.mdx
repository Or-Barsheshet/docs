## Creating a New Dataset

To create a new dataset, go to the **[Dataset Inventory](https://dash.readme.com/project/visual-layer2/v1.0/docs/understanding-clusters-copy)** view and click the "**+**" button or the "**New Dataset**" button.

![](https://files.readme.io/ca369b8add29da612e26061c671355ac052b2aae43419f79ba9432c80f001eb4-image.png)

### Dataset Name

You can enter a dataset name or use Visual Layer's auto-suggested name.

>  **Note:** Avoid using duplicate dataset names.

![](https://files.readme.io/4ab83c6e298ec22e1358a479bf4faa1115607dbafed628ba1b621fa76f4b25c4-image.png)

### Select Data Source

Select a source to upload data into your private dataset.  
**Supported data source types include:** 

- ZIP file
- Files / Folders 
- Public Amazon S3 
- Private Amazon S3

![](https://files.readme.io/3484a6682d53ec4fc182aa1d83deac9247e820deecd311a38e22c74c2b4f2ae3-image.png)

### Data Preview

The data preview is generated for the user to validate that the data source contains the required data.

- To replace the data source, press 'Edit' and follow the instructions.

![](https://files.readme.io/ca8f63b185bf1a6e7cd519f2f3142444a250f57a90a36fd99e987d13655e6c73-image.png)

***

## Adding Annotations - Optional

Import an annotation file to your dataset and press 'Next.' Visit [Upload Annotation](https://dash.readme.com/project/visual-layer2/v1.0/docs/importing-annotations-copy) for more details on supported formats. 

Click **"Skip"** if you don't want to upload annotations.

Data Preview with the annotations will be generated for you to validate that the data source is compatible with the annotations. 

![](https://files.readme.io/9a0e4cd3f9347a88f1e04cdc06e811550cc7ac88d39d672fe38f5a9c4c21f806-image.png)

If you wish to replace the annotations file, press 'Edit' and follow the instructions.

***

## Delete Dataset

To delete a dataset, navigate to Dataset Inventory, click the menu button next to the dataset you wish to delete, and press "Delete".

>  **Note:** Deleting a dataset is irreversible.
