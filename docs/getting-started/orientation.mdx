# Left Panel

The **Left Panel** provides quick navigation and access to additional resources and remains visible at all times.

![Left Panel](https://files.readme.io/888da66926a3df9280562a0556314d7bb9d4f796753c2b3fd394bae953afbabd-image.png)


 **My Datasets***My Datasets**: Click to return to the Dataset Inventory.  
 **Explore Public VL Dataset***Explore Public VL Dataset**: Browse publicly available datasets.  
 **Documentation***Documentation**: Access guides and reference materials.  
 **Bug Report***Bug Report**: Report issues or feedback.  
 **Version Info***Version Info**: View the current version of the application.  
 **Personal Info***Personal Info**: Manage account settings and preferences. clicking on this will open personal info, including image usage limits, subscription status, and logout options.
