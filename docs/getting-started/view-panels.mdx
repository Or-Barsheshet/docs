## Left Panel

The **Left Panel** provides quick navigation and access to additional resources and remains visible at all times.

<img src="https://files.readme.io/888da66926a3df9280562a0556314d7bb9d4f796753c2b3fd394bae953afbabd-image.png" alt="Left Panel" style={{ width: "8%" }} />

- **My Datasets**: Click to return to the Dataset Inventory  
- **Explore Public VL Dataset**: Browse public datasets  
- **Documentation**: Access guides and references  
- **Bug Report**: Report issues or feedback  
- **Version Info**: See current version  
- **Personal Info**: Manage account settings — includes usage limits, subscription status, and logout options  

_Default image quota: 10,000_  
Supports light/dark mode
