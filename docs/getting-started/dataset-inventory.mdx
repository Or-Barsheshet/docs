The **Dataset Inventory Page** provides a comprehensive **overview of your available datasets**, helping you easily explore, manage, and filter them.

The page displays key dataset details, such as:

![](https://files.readme.io/2b19bb734ff161026e40870c28ad6456cc4b1ec985afe29f398e5d82216ff02e-image.png)

Each dataset is **displayed as a card**, summarizing the dataset's key attributes.

- Dataset Name
- Creation Date
- Dataset Status (Ready, Processing, Failed, etc.)
- Dataset Type (Public/Private)
- Number of Images, Objects, Videos, and Video Frames

![](https://files.readme.io/5f8a6d5a8c118c48399ea06b4c527c7f840d55dd27cd14f42fe8c2cebc9efa66-image.png)

***

## Main Panel

The **Top Panel** contains essential tools for managing and navigating your datasets. It includes:

![](https://files.readme.io/6b4f29a4a181c27a911597223399aa6ea925ac6ec92bd74bacfdf0ee17251617-image.png)

- **Search Bar**: Locate datasets by name.
- **Dataset Type Filter**: Toggle between Public, Private, or All Datasets.
- **Sort By**: Arrange datasets based on status, creation date, or other attributes.
- **Clear Selection**: Reset applied filters.
- **New Dataset Button**: Create a new dataset. For more details, visit the Dataset Creation Guide.
