# Visual Layer Highly Secured Cloud

Visual Layer Cloud is a fully managed platform hosted on **AWS Cloud Services**, offering:

- **Secure** multi-tenant infrastructure
- **Fast onboarding **– Start with a trial and seamlessly scale to production
- **Cost-efficient pricing**
- **Latest features and enhancements** – Delivered through automatic updates

This option is ideal for teams looking to to **get started quickly** and for organizations with **strict governance requirements**, such as data residency and advanced security policies.

***

# On-Premises

Visual Layer’s Self-Managed deployment lets you run the platform entirely on your own infrastructure. 

- **Self-hosted** – Runs fully within your  private environment
- **Packaged solution** – Comes as a fully self-contained, easy-to-deploy package
- **Full data control **– No external dependencies. Your data stays within your infrastructure

This option is ideal for teams that require** full control over their data and environments**.
