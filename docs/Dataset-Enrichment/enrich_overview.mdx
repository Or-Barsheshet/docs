# Overview

Enriching your dataset is a 2-step flow:

1. Select Models  
2. Enrich Dataset  

---

## **Pre-requisites**

### Verify dataset is in the READY state:

1. Navigate to the dataset view and confirm that the dataset status is "READY". Alternatively, go to the dataset inventory, locate your dataset and verify its status.

## Model Selection

1. Navigate to Enrich tab, select a model and click on "Generate Models Preview" button.

<img 
  src="https://files.readme.io/48d302b7b8975a547fc20895f2261373080d920b4f2f0ce32c3daebb0148bbd0-image.png" 
  alt="Enrich: Select Model to generate preview" 
  style={{ display: "block", margin: "0 auto", border: "1px solid #ccc" }} 
/>

2. Add additional models by clicking on Add Models (optional) 

<img 
  src="https://files.readme.io/4af51591dd776faa73acd1bd869415127750bc4afa8b5e991d62365609a7b58a-image.png" 
  alt="Enrich: Add Additional Models for Preview" 
  style={{ display: "block", margin: "0 auto", border: "1px solid #ccc" }} 
/>

2. Navigate between models to review the model output before enriching the entire dataset. 

<img 
  src="https://files.readme.io/935f0fd46f8a22191a17bdee3c294b4abc67ed9a0f057669633e759ae2e7734f-image.png" 
  alt="Enrich: Click on Model Name for view Model Run Preview" 
  style={{ display: "block", margin: "0 auto", border: "1px solid #ccc" }} 
/>

2. To initiate the model run, click on "Enrich Dataset".  
   1. All the images/frames in the dataset will be enriched by the models listed in the Models section.  
   2. This operation may take some time, depending on the size of the dataset.  

3. Track enrichment progress  
   1. You can track progress by navigating to the Enrich tab in the dataset view. Alternatively you can track progress by navigating to the Dataset Inventory view - Notifications.  
   2. Notification will be displayed once process completes.  
   3. Note: User can explore their dataset throughout the enrichment process.

  style={{ display: "block", margin: "0 auto", border: "1px solid #ccc" }} 
/>

4. Explore Enriched Dataset  

   1. Select Exploration View - Images (default) or Objects.  
      1. Expand the VL Enrichment section in the right pane to view the metadata added to the dataset.

<img 
  src="https://files.readme.io/2a5ab60bf055a614b7e94fdfb7ed99bb9d321ce3437efcc32873266137c1b79e-image.png" 
  alt="Explore Enriched Dataset: VL enrichment Section in right-pane" 
  style={{ display: "block", margin: "0 auto", border: "1px solid #ccc" }} 
/>

   1. Select metadata type to display on cluster card:  
      1. Labels  
      2. Captions  
      3. User Tags (not generated by models but by Users. See [User Tags section](https://docs.visual-layer.com/docs/data-selection-using-visual-layer)).
